Our Mission

To provide a more productive Youtube experience.

About

YouTools provides a utilities and productivity platform for people to enhance thier experience of using Youtube. The app is a growing set of productivity features that users can use to get the best out of thier time spent on youtube. A couple of such features are:

  • Search word feature: Here the user can load the youtube video of thier choice by searching the youtube url in the search bar. Once the video is loaded in the resulting screen, if the video supports manual captions, they can search for a specific word in the video. This search will later result in a set of timestamps where the word is said in the video. Now the user can click to navigate directly to that section of the video specified in the timestamp.
  • Bookmark with Notes: This feature is still under development. Here the user can choose to add bookmarks in the view of thier choice. And each bookmark can be added with "Notes". These bookmarks are saved for the the specific user after they login (having signed up). And can be retried from a saved session of the video.

YouTools will in the course of time be added with features that will make a users youtube experience productive. This will mainly be usefull for students or others wanting to learn somthing in a systematic manner.

Users of this app can make use of the "Contact" section on this website to let me know of some of the features they would like to see implemented in this tool.

The application is built entirely on various javascript frameworks, namely Angular, Node, Express, Mongo DB etc. It utilizes api's provided by Youtube. The application will grow in its features with time, and try operate indepenet of the api's it currently uses, owing to the fact that they are being depricated.
